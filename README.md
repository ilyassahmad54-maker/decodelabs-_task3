# 🩺 Diabetes Prediction System using Machine Learning

## 📌 Project Overview

This project demonstrates a complete end-to-end Data Science and Machine Learning workflow for predicting diabetes based on medical diagnostic features. It includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, and deployment using a Streamlit web application.

The goal is to build a system that can predict the likelihood of diabetes in patients and support early detection, which is especially important for improving healthcare outcomes in countries like Pakistan.

---

## 🚀 Features

- Data cleaning and preprocessing
- Handling missing and invalid values
- Exploratory Data Analysis (EDA) with visualizations
- Machine Learning model (Logistic Regression)
- Model evaluation using accuracy and confusion matrix
- Interactive Streamlit web app for real-time predictions

---

## 📊 Dataset

The dataset contains medical features such as:
- Glucose level
- Blood pressure
- BMI
- Age
- Insulin levels
- Skin thickness

These features are used to predict whether a patient is diabetic or not.

---

## 🧠 Machine Learning Model

- Algorithm used: **Logistic Regression**
- Type: Binary Classification
- Output: Diabetic / Non-Diabetic prediction

### Evaluation Metrics:
- Accuracy Score
- Confusion Matrix
- Model Performance Analysis

---

## 🖥️ Streamlit Web App
